# circle_bulletin_board
サークル掲示板サイト

 
### API起動方法
```
.
├── circle_bulletin_board_api
│   ├── __init__.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
└── manage.py
```
このディレクトリで```python manage.py runserver```で起動する
Python 3.6.0
Django 1.11.29
### Frontの起動方法
frontednディレクトリで```npm run start```で起動可能
